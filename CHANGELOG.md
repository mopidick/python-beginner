# Changelog

## v0.3.2

- 进度存储升级到 schema v2，记录每关练习次数、最后练习时间和首次通关时间，旧版 localStorage 会自动迁移。
- 学习概览把“已尝试”升级为“练习次数”，并新增可点击的复习队列，最多展示 3 个收口或回刷入口。
- 复习候选按未通关尝试和尝试次数排序，多次卡住的关卡会优先排在队列前面。
- 抽离 `progress/storage.ts`，把进度读写和迁移从主应用组件中拆出，便于后续继续做复习闭环。
- 新增 progress v1 -> v2 迁移、复习排序和复习队列展示测试。

## v0.3.1

- 修复课程内容中的 `????` 占位和不可读提示，补齐新增复习关、布尔关、空列表兜底关和项目关的中文教学文案。
- 关卡地图新增“新课 / 复习 / 项目”标签，让用户能区分当前学习阶段。
- 学习概览新增“薄弱知识点”摘要，基于未通关尝试、低星通关和提示使用聚合复习方向。
- 项目关增加多检查点，数据清洗项目同时检查 `names` 和 `scores`，成绩报告项目同时检查 `passed_count`、`average` 和 `report`。
- 新增内容可读性、项目关检查点、薄弱知识点和版本一致性门禁，防止坏文案和版本漂移进入发布。

## v0.3.0

- 新增继续学习建议和复习推荐入口，学习概览会根据当前进度推荐下一关、卡住关或低星复习关。
- 课程从 32 关扩展到 40 关，新增复习关、布尔判断、空值兜底、CSV 风格清洗和成绩报告项目。
- 关卡数据增加 `mode` 和 `tags`，区分 lesson、review、project，课程地图和推荐逻辑有更清晰的学习语义。
- 新增版本一致性、推荐算法、内容元数据等发布门禁测试。
- README、前端页面、后端健康检查和前端包版本统一升级到 v0.3.0。

## v0.2.5 Experience Patch

不提升版本号，集中增强用户可感知体验。

- 课程从 15 关扩展到 32 关，新增数值运算、字符串清洗、条件 Boss、列表追加/排序、循环搜索、字典计数、函数参数/返回值、计算器、单词统计、学生成绩管理等内容。
- 每关增加章节、难度、预计时间和任务故事，关卡列表升级为章节化课程地图。
- 学习概览增加章节数和预计总时长，帮助用户看到完整学习路径。
- 通关后增加 1-3 星反馈，无提示通关可获得 3 星，增强回刷动机。
- 新增课程内容测试，要求课程至少 30 关、至少 6 个章节，并验证每关学习元数据完整。
- 关卡配置迁移到 `shared/levels.json` 单一数据源，前端课程地图和后端检查规则从同一份结构化内容读取。
- 修复数值检查的浮点容差问题，避免 `100 * 1.1` 这类合法答案被误判失败。
- 失败检查返回当前值、目标值和原因说明，前端新增“本次要修正”区域，让用户直接看到差在哪里。
- 课程地图按章节分组展示完成数，并在已通关关卡显示星级，学习路径更清楚。
- 切换关卡时自动保留未运行草稿；清空进度前增加确认，避免误删本地学习记录。
- 每关补充学习目标、解题套路和通关复盘，让 32 关更像一套课程而不只是题库。

## v0.2.5

新增学习概览：展示已通关、已尝试、已用提示、当前关卡，并支持清空本地进度。

## v0.2.4

课程从 10 关扩展到 15 关，新增字符串方法、列表推导式、嵌套数据、模拟文件行处理和第二个综合挑战。

## v0.2.3

变量快照升级为结构化可视化。列表按索引展示，字典按 key/value 展示，标量值以代码样式展示。

## v0.2.2

新增每关 3 层递进提示，并把提示使用数写入 localStorage，方便后续复盘。

## v0.2.1

后端 `/api/run` 增加 `diagnostics` 字段，把 SyntaxError、NameError、TypeError、ZeroDivisionError、TimeoutError、ImportError 等错误映射为中文解释和修复建议；前端状态面板展示“诊断反馈”。

## v0.2.0

编辑器从 textarea 升级到 CodeMirror 6，支持 Python 语法高亮、行号、括号匹配、自动缩进、基础补全和 Ctrl/Command + Enter 运行。新增补全核心测试和编辑器组件测试。

## v0.1.1

补丁迭代，增强反馈闭环和接口可靠性。

- 后端新增 `/api/health` 和 `/api/levels`。
- `/api/run` 增加 Pydantic 响应模型。
- 检查结果增加 `hint` 字段。
- 状态面板增加运行摘要。
- 关卡列表增加“尝试中”状态。
- 前端 HTTP 错误提示改为稳定中文文案。

## v0.1.0

首个可运行 MVP。

- React + TypeScript + Vite 前端学习界面。
- FastAPI 后端 `/api/run` 执行接口。
- 子进程执行 Python，捕获 stdout、stderr、异常和变量快照。
- 10 个核心语法关卡。
- localStorage 保存通关进度和最近代码。
